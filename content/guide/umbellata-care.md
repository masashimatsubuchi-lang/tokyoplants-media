# ウンベラータの育て方｜葉が落ちる原因と対策

ハート型の大きな葉と白い幹が美しいウンベラータは、インテリアグリーンとして非常に人気の高い観葉植物です。おしゃれなカフェやショップでもよく見かける一方で、「冬に葉が全部落ちてしまった」「買ったときより元気がない」という声も少なくありません。しかし、ウンベラータは環境さえ整えれば丈夫に育つ植物です。この記事では、水やり・置き場所・剪定・冬越しのコツと、よくあるトラブルへの対処法を解説します。

## ウンベラータの基本情報

| 項目 | 内容 |
|------|------|
| 学名 | Ficus umbellata |
| 科・属 | クワ科フィカス属 |
| 原産地 | 熱帯アフリカ |
| 耐寒温度 | 約10℃ |
| 成長速度 | 速い（年間20〜30cm伸びることも） |

ウンベラータはゴムの木の仲間です。枝や葉を切ると白い樹液が出ますが、この樹液は肌に付くとかぶれる場合があります。剪定時はゴム手袋を着用し、床や家具に新聞紙を敷いておくと安心です。

## 置き場所

ウンベラータは日光を好む植物です。室内であれば、レースカーテン越しの明るい窓際がベストな置き場所になります。

**押さえておきたいポイント**

- 直射日光は葉焼けの原因になるため、真夏は遮光するかレースカーテン越しに置く
- 暗い場所に置くと葉が小さくなり、間延びした樹形になりやすい
- エアコンや暖房の風が直接当たる場所は避ける（乾燥で葉が傷む）
- 春から秋にかけて、たまに屋外の半日陰に出すと生育が良くなる

急に環境を変えると葉を落とすことがあります。置き場所を移動する場合は、1週間ほどかけて徐々に光量を調整してください。

## 水やり

ウンベラータの水やりは季節によって頻度を変えるのが基本です。

| 季節 | 頻度の目安 | ポイント |
|------|-----------|---------|
| 春（4〜6月） | 土の表面が乾いたら | 成長期に入るため、しっかり水を与える |
| 夏（7〜9月） | 土の表面が乾いたら | 鉢底から水が流れ出るまでたっぷり与える |
| 秋（10〜11月） | 土の表面が乾いて2日後 | 徐々に水やりの間隔を空ける |
| 冬（12〜3月） | 土が完全に乾いてから | 控えめに。月2〜3回程度が目安 |

水やりの共通ルールとして、受け皿に溜まった水は必ず捨ててください。根が常に水に浸かった状態は根腐れの原因になります。また、冬場は水やりを控える代わりに、霧吹きで葉に水をかける「葉水」を行うと、乾燥対策とハダニ予防に効果的です。

## 剪定と樹形づくり

ウンベラータは成長が速いため、定期的な剪定で好みの樹形に仕立てることができます。

**剪定の時期と方法**

- 適期は5〜9月の成長期。この時期なら切った箇所の下から新芽が出やすい
- 切りたい位置の節（葉が付いていた跡）のすぐ上で切る
- 切り口から白い樹液が出るため、ティッシュで押さえて拭き取る
- 太い枝を切った場合は、癒合剤を塗ると回復が早い

枝分かれさせたい場合は、分岐させたい高さで思い切って幹を切ります。切り口付近から2〜3本の新芽が出て、枝分かれした樹形をつくることができます。また、剪定で出た枝は水挿しや取り木で増やすことも可能です。

## 冬の管理

ウンベラータにとって冬は最も注意が必要な季節です。熱帯アフリカ原産のため寒さには弱く、室温が10℃を下回ると葉を落とし始めます。

**冬越しのポイント**

- 最低でも10℃以上をキープする。理想は15℃以上
- 窓際は夜間に冷え込むため、夜は部屋の中央寄りに移動させる
- 水やりは控えめにし、土が完全に乾いてから与える
- 暖房による乾燥対策として、こまめに葉水を行う

冬に葉がすべて落ちてしまっても、幹が緑色でしっかりしていれば枯れていない可能性が高いです。春になり気温が上がれば、新芽が出て復活するケースが多いので、諦めずに管理を続けてください。

## よくあるトラブルと対策

| トラブル | 主な原因 | 対策 |
|---------|---------|------|
| 葉が落ちる | 環境の急変、寒さ、水切れ | 置き場所を安定させる。冬は10℃以上を保つ |
| 葉が黄色くなる | 根腐れ、日照不足 | 水やり頻度を見直す。明るい場所に移動する |
| 葉が小さくなる | 日照不足、根詰まり | 明るい場所に置く。鉢が窮屈なら植え替える |
| ハダニが付く | 乾燥した環境 | 葉水をこまめに行う。被害が広がったら薬剤を使用 |

特にハダニは乾燥する冬場に発生しやすい害虫です。葉の裏に小さな点々や薄い糸が見えたら、すぐにシャワーで葉の裏表を洗い流してください。日頃から葉水を習慣にしておくと予防になります。

## まとめ

ウンベラータは「明るい場所」「適切な水やり」「冬の保温」の3つを押さえれば、初心者でも長く楽しめる観葉植物です。冬に落葉しても春に復活する生命力を持っているので、慌てずに管理を続けることが大切です。成長が速く剪定にも強いため、自分好みの樹形に育てる楽しさも味わえます。

植え替えの際は、水はけの良い土を選ぶことが健康な根を保つポイントです。
